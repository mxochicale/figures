Once you create a drawing, you can convert it easly in the terminal 
using the following commands!


```
inkscape --export-png drawing_v##.png drawing.svg
inkscape --export-pdf $(@) $(<)
inkscape -z -e drawing.png -w 1024 -h 1024 drawing.svg
```
